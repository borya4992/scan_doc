# Scan Doc — сканер документов в Excel

Бесплатное мобильное веб-приложение для сканирования **PDF** и **JPG** с экспортом в Excel по вашему шаблону. Работает на **Vercel** (бесплатный тариф).

## Возможности

- Сканирование с камеры телефона или загрузка файлов (PDF, JPG, PNG)
- OCR на русском языке (Tesseract.js) — обработка на телефоне, без серверных лимитов
- Автоматическое извлечение полей: Сотрудик, Табель номер, Должность, Конец периода, День
- Пакетная обработка 500+ документов в день
- Сохранение результатов в IndexedDB (данные остаются на устройстве)
- Экспорт в Excel по шаблону `public/template.xlsx`
- PWA — можно добавить на главный экран телефона

## Запуск без терминала (Windows)

Приложение можно запускать **в фоне** — окно терминала не нужно.

### Быстрый запуск

Дважды щёлкните файл **`Start Scan Doc.vbs`** в папке проекта.

Откройте http://localhost:3000 (или с телефона: http://ВАШ_IP:3000).

### Остановка

Дважды щёлкните **`Stop Scan Doc.vbs`**.

### Автозапуск при включении Windows

```powershell
powershell -ExecutionPolicy Bypass -File "d:\проекты\Scan_doc\scripts\install-autostart.ps1"
```

После этого приложение будет стартовать само при загрузке Windows.

### Через npm

```bash
npm run start:bg   # запуск в фоне
npm run stop:bg    # остановка
```

## Облачный запуск (Vercel) — работает всегда, с любого телефона

Лучший вариант: приложение на сервере Vercel — **не нужен компьютер и терминал**.

1. Зарегистрируйтесь на [vercel.com](https://vercel.com) (бесплатно)
2. Нажмите **Add New** → **Project**
3. Загрузите папку проекта через GitHub **или** установите Vercel CLI:

```bash
npx vercel login
npx vercel --prod
```

4. Получите постоянную ссылку вида `https://scan-doc.vercel.app` — открывайте её с телефона из любой сети.

## Локальный запуск (для разработки)

```bash
npm install
npm run dev
```

Откройте http://localhost:3000 на телефоне (в той же Wi‑Fi сети) или в браузере.

## Деплой на Vercel (бесплатно)

1. Загрузите проект на GitHub
2. Зайдите на [vercel.com](https://vercel.com) → **Add New Project**
3. Импортируйте репозиторий
4. Нажмите **Deploy** — настройки по умолчанию подходят

Или через CLI:

```bash
npm i -g vercel
vercel
```

## Как пользоваться

1. Откройте приложение на телефоне
2. Нажмите **Сфотографировать** или **Выбрать файлы**
3. Дождитесь обработки (можно загружать много файлов сразу)
4. При необходимости отредактируйте поля вручную
5. Нажмите **Экспорт в Excel**

## Важно для 500+ документов в день

- OCR выполняется **на телефоне** — это бесплатно и не упирается в лимиты Vercel
- Обрабатывается по 2 файла параллельно, чтобы не перегружать память
- Все результаты сохраняются локально — можно закрыть браузер и продолжить позже
- Для лучшего качества: хорошее освещение, документ без бликов

## Замена шаблона Excel

Замените файл `public/template.xlsx` на свой. Первая строка должна содержать заголовки:

`№`, `Сотрудик`, `Табель номер`, `Должность`, `Конец периода`, `День`
